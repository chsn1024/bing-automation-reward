# MS-Rewards-Bot 🌸

一个基于 [Playwright](https://playwright.dev/python/) 的微软搜索奖励（Microsoft Rewards）自动化脚本集合。  
**目前仅供学习与研究，滥用可能导致账号异常或封禁，请谨慎使用！**

---

## ✨ 已实现功能
- 自动登录微软账户并保存 Cookie  
- 随机读取关键词，模拟真人输入、滚动、点击  
- 每日任务中心卡片自动点开并浏览  
- 所有操作都加了随机延时与鼠标轨迹，降低机器特征

---

## ⚠️ 已知问题
1. **易被检测**  
   尽管做了去自动化特征、随机延时、鼠标轨迹等伪装，仍有概率被微软识别为机器人，导致积分不予发放甚至账号异常。  
   *短期内没有 100 % 可行方案，欢迎提 Issue 或 PR 交流。*

2. **每次搜索都会回到首页**  
   当前逻辑是：搜索 → 滚轮浏览 → 回退到首页 → 再搜索。  
   这样会增加请求量，也容易触发风控。
